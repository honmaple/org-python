* org-python
  An orgmode parser for converting orgmode to html based on python.

  [[https://pypi.python.org/pypi/org-python][https://img.shields.io/badge/pypi-v0.2.9-brightgreen.svg]]
  [[https://python.org][https://img.shields.io/badge/python-3-brightgreen.svg]]
  [[LICENSE][https://img.shields.io/badge/license-BSD-blue.svg]]

** quickstart
   #+BEGIN_SRC sh
  pip install org-python
   #+END_SRC

   #+BEGIN_SRC python
     from orgpython import to_html

     text = '''* heading
     - list1
     - list2
     - list3
       - list4
     - list5

       | th1-1  | th1-2  | th1-3  |
       |--------+--------+--------|
       | row1-1 | row1-2 | row1-3 |
       | row2-1 | row2-2 | row2-3 |
       | row3-1 | row3-2 | row3-3 |
     '''
     print(to_html(text, toc=True, offset=0, highlight=True))
   #+END_SRC
